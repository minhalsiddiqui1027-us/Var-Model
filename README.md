### Overview
This project implements a Value at Risk (VaR) based financial risk model using real market data.

### Features
- Fetches historical stock and market data
- Calculates daily returns
- Computes Beta using covariance and variance
- Estimates required return using CAPM
- Calculates Exposure at Risk (EaR)

### Tools Used
- Python
- Jupyter Notebook
- yfinance, pandas, numpy
